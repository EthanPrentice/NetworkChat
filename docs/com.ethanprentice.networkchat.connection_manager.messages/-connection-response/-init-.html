ConnectionResponse(ip: String, port: Int, accepted: Boolean, tcpPort: Int?, groupInfo: GroupInfo?)

Used to to be sent to a UDP socket on a device that sent a ConnectionRequest

Parameters

ip - The ip address of the current device

port - The port that the UDP socket is running on on the current device

accepted - Whether the ConnectionRequest was accepted by the current user or not

tcpPort - The TCP port for the client to connect to if accepted (null when accepted is false)
